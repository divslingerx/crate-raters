# CrateRaters Modernization Design

## Stack

- Express 5 + EJS with ejs-mate for layouts
- PostgreSQL via Docker Compose (replacing MongoDB)
- Drizzle ORM (replacing Mongoose) — uses generated schema, no hand-written SQL
- Better Auth (replacing Passport) — generates its own auth tables
- TypeScript with tsx for dev
- Vite as build tool for Tailwind CSS + client-side JS
- HTMX for dynamic interactions

## Project Structure

```
src/
  app.ts              # entry point
  db/
    index.ts          # drizzle client
    schema.ts         # drizzle table definitions (records, comments)
  routes/
    index.ts          # landing
    records.ts        # CRUD
    comments.ts       # CRUD
    auth.ts           # better-auth routes
  middleware/
    index.ts          # ownership checks, isLoggedIn
  client/
    main.css          # tailwind entry
views/
  layouts/
    main.ejs          # base layout (ejs-mate)
  records/            # index, show, new, edit
  comments/           # new, edit
  landing.ejs
  login.ejs
  register.ejs
public/               # vite outputs built CSS here
vite.config.ts
tailwind.config.ts
drizzle.config.ts
tsconfig.json
```

## Database Schema

- Auth tables (users, sessions, accounts) — generated by Better Auth
- `records` — id, artist, album, image, year, found, userId (FK to auth users)
- `comments` — id, text, userId (FK), recordId (FK), createdAt

## Auth

Better Auth handles registration, login, logout, sessions. Mounted on `/api/auth/*`. Session middleware populates req.user equivalent.

## Frontend Build

Vite watches `src/client/main.css` (Tailwind entry), outputs to `public/dist/`. Express serves `public/` statically. HTMX loaded via CDN in the layout.

## Scripts

- `dev` — docker compose up, vite build --watch, tsx src/app.ts (concurrently)
- `build` — vite build (production CSS)
- `db:generate` — drizzle-kit generate
- `db:migrate` — drizzle-kit migrate
- `db:push` — drizzle-kit push (quick dev sync)
- `db:seed` — drizzle seed
- `test` — run test suite

## Removed

- Mongoose, all models/*.js
- Passport, passport-local, passport-local-mongoose, connect-flash
- All .js source files (replaced by .ts in src/)
- MongoDB from docker-compose

## Kept

- EJS views (updated for ejs-mate layouts + Tailwind classes)
- method-override
- Docker Compose (switched to Postgres)
- Seeds concept (using Drizzle seeding)
- Tests written alongside each feature
